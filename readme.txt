passoa para usar o git

1. na pasta raiz da aplicação executar o comando - git init
2. executar o comando - git config user.name "<nome do usuario>"
3. executar o comando - git config user.email <email da conta no git>
4. (opcional)
   executar o comando - git config - m "<observação>"
5. executar o comando - git add <.> ou <arquivo>   
   O (.) faz com que todos os arquivos sejam marcados para commit
   Mas, uma outra opção é identificar qual arquivo será marcados
6. executar o comando - git commit
